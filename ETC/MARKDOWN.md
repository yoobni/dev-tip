# Markdown
Text based markup language

#### Header
- Big header
```
Header
======
```
- Small header
```
Header
------
```

#### Title
```
# H1 title
## H2 title
...
##### H5 title
###### H6 title
```

#### Font design
- Bold
```
**bold font**
__bold font__
```
- Italic
```
*italic font*
_italic font_
```
- Delete
```
~~delete font~~
```

#### List
- Number list
```
1. one
2. two
3. three
```
- Non-numeric list
```
* apple
    * banana
        * cherry

+ apple
    + banana
        + cherry

- apple
    - banana
        - cherry
```

#### Line
```
* * *
***
*****
- - -
---
-----
------------------
```

#### Link
- Auto link
```
(linkURL)
```
- Inline link
```
[link keyword](url)
```
- Image
```
![alt text](/path/image.jpg)
<img src="" alt="">
```

#### BlockQuote
```
> block quote
```

#### Code
```
    ```
        console.log
    ```
```
